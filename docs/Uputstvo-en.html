Guide to MetaMask and BNB Smart Chain

1. Installing the MetaMask Crypto Wallet

Download and install MetaMask for your device:

  • iOS (App Store)
  • Android (Google Play)
  • Chrome / Brave Extension (Windows/Mac)
  • Official Website for All Platforms

2. Adding the BNB Smart Chain Network

Open MetaMask, go to Settings → Networks → Add Network. You can add the BNB Smart Chain automatically via Chainlist by clicking “Add to MetaMask,” or enter the following parameters manually (source: Binance Academy):

  • Network Name: BNB Smart Chain
  • New RPC URL: https://bsc-dataseed.binance.org/
  • Chain ID: 56
  • Currency Symbol: BNB
  • Block Explorer URL: https://bscscan.com

3. Adding the Dušanov Zlatnik Token

After switching MetaMask to the BNB Smart Chain network, click on the "Import Tokens" button and manually enter the following details:

  • Token Contract Address: 0x27a81dce6f4bd0f2a6f3c17e195ecdb773051a00
  • Symbol: ZLATNIK
  • Decimals: 18

You can verify the information on the official BscScan token profile.

Once successfully added, the Dušanov Zlatnik token will appear in your MetaMask wallet.
