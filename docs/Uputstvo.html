Како додати Душанов Златник у MetaMask

1. Инсталација MetaMask-а

Преузмите и инсталирајте MetaMask према уређају који користите:

  • iOS (App Store)
  • Android (Google Play)
  • Chrome / Brave екстензија (Windows/Mac)
  • Званични сајт за све платформе

2. Додавање BNB Smart Chain мреже

Отворите MetaMask → Подешавања → Мреже → Додај мрежу BNB Smart Chain преко Chainlist, или ручно унесите следеће параметре (извор: Binance Academy):

  • Network Name: BNB Smart Chain
  • New RPC URL: https://bsc-dataseed.binance.org/
  • Chain ID: 56
  • Currency Symbol: BNB
  • Block Explorer URL: https://bscscan.com

Након што сте на BNB мрежи, кликните „Увези токен“ и ручно унесите:

  • Token Contract Address: 0x27a81dce6f4bd0f2a6f3c17e195ecdb773051a00
  • Symbol: ZLATNIK
  • Decimals: 18

Можете проверити податке и на званичном BscScan профилу токена.

По завршетку, Душанов Златник ће бити видљив у вашем новчанику.

How to Add Dušanov Zlatnik to MetaMask

1. Installing MetaMask

Download and install MetaMask for your device:

  • iOS (App Store)
  • Android (Google Play)
  • Chrome / Brave Extension (Windows/Mac)
  • Official Website for All Platforms

2. Adding the BNB Smart Chain Network

Open MetaMask → Settings → Networks → Add BNB Smart Chain network via Chainlist, or manually enter the following parameters (source: Binance Academy):

  • Network Name: BNB Smart Chain
  • New RPC URL: https://bsc-dataseed.binance.org/
  • Chain ID: 56
  • Currency Symbol: BNB
  • Block Explorer URL: https://bscscan.com

Once connected to the BNB network, click "Import Token" and manually enter:

  • Token Contract Address: 0x27a81dce6f4bd0f2a6f3c17e195ecdb773051a00
  • Symbol: ZLATNIK
  • Decimals: 18

You can also verify the details on the official BscScan token profile.

After completing these steps, Dušanov Zlatnik will be visible in your wallet.
