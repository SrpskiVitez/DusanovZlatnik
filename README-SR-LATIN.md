
# Душанов Златник — Projekat i Web sajt

Dobrodošli u zvanični repozitorijum projekta **Душанов Златник**!

---

## Opis projekta

**Душанов Златник** je kripto-token i edukativni projekat sa srpsko-nacionalnom tematikom, inspirisan istorijskim likom cara Stefana Dušana Silnog. Projekat kombinuje istorijske sadržaje, pravne dokumente, i kripto tehnologiju kako bi promovisao srpsku tradiciju i kulturu u digitalnom svetu.

Na zvaničnom sajtu dostupni su detaljni tekstovi o:

- Istoriji cara Dušana i njegovom nasleđu
- Pravilima korišćenja tokena (Terms of Use)
- Odricanju od odgovornosti (Disclaimer)
- Politikama privatnosti (Privacy Policy)
- Statusu utility tokena (Utility Status)
- Belom papiru / tehničkim informacijama (Whitepaper)

---

## Struktura sajta

- `/index.html` — Početna stranica projekta
- `/docs/terms.html` — Pravni uslovi korišćenja
- `/docs/disclaimer.html` — Odricanje od odgovornosti
- `/docs/privacy.html` — Politika privatnosti
- `/docs/utility-status.html` — Status utility tokena
- `/docs/whitepaper.html` — Beli papir sa tehničkim detaljima
- `/pic/` — Slike i ilustracije
- `/style-docs.css` — CSS stilovi za dokumenta
- `/scr/script-docs.js` — JavaScript za interaktivnost (npr. promena jezika)

---

## Jezici podrške

Sajt je dostupan na tri jezika:

- Српска ћирилица (`sr`)
- Српска латиница (`sr-latin`)
- Engleski (`en`)

Jezik se može menjati preko jezičkog switchera na svakoj stranici.

---

## Tehničke karakteristike

- Izgrađen kao statički sajt hostovan na GitHub Pages
- backend api ,php, hostovan na https://dusanov-api.vercel.app/api/send
- Korišćenje modernog HTML5 i CSS3 standarda
- Content Security Policy (CSP) za veću bezbednost sajta
- Minimalna upotreba JavaScript-a, uglavnom za jezičke funkcionalnosti i osnovnu interakciju
- Resursi (ikone, slike, fontovi) se učitavaju sa sopstvenog servera ili sigurnih CDN-ova
